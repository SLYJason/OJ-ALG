# 215: Kth Largest Element in an Array

### Solution: Min Heap
**Time Complexity:** `O(N * logk)`.

**Space Complexity:** `O(k)`.

### Solution 2: Quick Select
Using quick sort logic here.

**Time Complexity:** `O(N^2)`. The worst case for the quick sort is `O(N^2)`.

**Time Complexity:** `O(1)`.

# 322: Coin Change

### Solution 1: Bottom-Up DP
**Time Complexity:** `O(N) * A`. `N` is the length of `coins`, `A` is the value of `amount`.

**Space Complexity:** `O(A)`.

### Solution 2: Top-Down DP
**Time Complexity:** `O(N) * A`.

**Space Complexity:** `O(A)`.

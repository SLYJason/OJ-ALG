# 84: Largest Rectangle in Histogram
 
### Solution 1: Brute Force
For each `height[i]`, go right direction to find `min_height`, in the meantime update the `max_area`. 

**Time Complexity:** `O(N ^ 2)`.
 
**Space Complexity:** `O(1)`.

# 104: Maximum Depth of Binary Tree

### Solution 1: BFS
Time Complexity: `O(N)`, N is the number of nodes.

Space Complexity: `O(N)`, N is the number of nodes.

### Solution 2: DFS, Bottom-Up
Time Complexity: `O(N)`, N is the number of nodes.

Space Complexity: `O(N)`, N is the number of nodes.

### Solution 3: DFS, Tom-Down
Time Complexity: `O(N)`, N is the number of nodes.

Space Complexity: `O(N)`, N is the number of nodes.
